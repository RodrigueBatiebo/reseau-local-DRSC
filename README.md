# reseau-local-DRSC
## Description
Simulation du réseau local de la Direction Régionale 
de la Santé du Centre (Burkina Faso), réalisée dans 
le cadre d'un stage à l'ANPTIC.

## Architecture
- SW-L3 : Commutateur de niveau 3
- SW-R0, SW-R1 : Commutateurs d'accès
- AP1, AP2 : Points d'accès Wi-Fi
- VLAN10 (VoIP), VLAN20 (PC-agent), VLAN30 (Wi-Fi)

## Technologies utilisées
- GNS3
- Cisco IOS (switches L2/L3)
- VLANs, EtherChannel/LACP, ACL
